# Hello OPN Builder

This repository is my first Web3 project for the OPN Builder Program.

## Features

- Solidity Smart Contract
- Hardhat Project
- Deployment Script
- Unit Tests

## Tech Stack

- Solidity
- Hardhat
- Ethers.js

## Commands

Install

```bash
npm install
```

Compile

```bash
npx hardhat compile
```

Test

```bash
npx hardhat test
```

Deploy

```bash
npx hardhat run scripts/deploy.js
```
